## Expense Tracker Client

### Stack

* React
* Redux

### Run locally

* Clone this repo
* Ensure `yarn` is installed
* Run `yarn install` to get deps
* Run `yarn start`
* Navigate to `localhost:5000`

### Run with docker

* Ensure docker is installed
* Run `docker build -t <image-name> .`
* Run `docker run -p 5000:5000 -d <image-name>`
* Navigate to `localhost:5000`
